# 大学生家教平台

这是一个基于Spring Boot和Vue的大学生家教平台，旨在为学生和家教提供一个方便的沟通和管理平台。

## 功能

- 学生可以在平台上发布求教信息
- 家教可以在平台上申请接单
- 家教和学生可以通过平台进行沟通
- 提供课程管理、评价系统等功能

## 技术栈

- **后端**: Spring Boot
- **前端**: Vue.js
- **数据库**: MySQL
- **其他**: MyBatis, Spring Security, JWT, Axios
